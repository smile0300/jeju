# 제주 여행 도우미 데이터 연동 가이드 (Data Info Guide)

이 프로젝트에서 각각의 실시간 정보를 어떻게 불러오는지에 대한 기술적 상세 가이드입니다.

## 1. 실시간 CCTV (📹)
- **방법**: 공공기관에서 제공하는 HLS 방식의 스트리밍 URL을 직접 연결하거나 YouTube 라이브 스트리밍을 임베드합니다.
- **기술**: 
  - `Hls.js`: 일반 브라우저에서 HLS(.m3u8) 영상을 재생하기 위해 사용합니다.
  - `CORS Proxy`: 보안 정책(CORS)으로 인해 직접 접근이 불가능한 스트리밍 소스는 자체 Cloudflare Worker 프록시(`PROXY_URL`)를 통해 우회하여 불러옵니다.
  - `YouTube IFrame API`: 유튜브 라이브 채널의 경우 IFrame을 통해 자동 재생 및 무음 설정을 적용합니다.

## 2. 날씨 정보 (🌤️)
- **방법**: **기상청 단기예보 및 중기예보 API**를 활용합니다.
- **출처**: [공공데이터포털 (data.go.kr)](https://www.data.go.kr)
- **상세**:
  - `getVilageFcst` (단기예보): 현재 기온, 강수 확률, 습도, 풍속 등을 3시간 단위로 가져옵니다.
  - `getWthrWrnMsg` (기상특보): 제주 지역(stnId: 184)에 발효된 풍랑, 강풍, 호우 등의 실시간 특보를 가져옵니다.

## 3. 한라산 등반 통제 (⛰️)
- **방법**: **제주특별자치도청 홈페이지**의 실시간 정보를 데이터 파싱(Scraping)하여 가져오거나 관련 API를 활용합니다.
- **특이사항**: CORS 정책 우회를 위해 프록시 서버를 거쳐 HTML을 분석하고, 탐방로별(어리목, 영실 등) 통제 상태를 추출합니다.

## 4. 항공편 운항 정보 (✈️)
- **방법**: **한국공항공사 실시간 운항 정보 API**를 연동합니다.
- **상세**:
  - `getArrFlightStatusList` (도착): 제주공항(CJU)으로 들어오는 항공편 상태를 조회합니다.
  - `getDepFlightStatusList` (출발): 제주공항에서 나가는 항공편 상태를 조회합니다.
  - 중화권 및 국내 주요 공항 코드를 필터링하여 사용자에게 필요한 정보만 노출합니다.

## 🎬 축제 및 행사 (Festivals & Events)
- **제공처**: 비짓제주 (Visit Jeju)
- **방식**: OpenAPI 및 공식 포털 API 활용
- **상세 내용**:
    - **Endpoint**: `https://api.visitjeju.net/vsjApi/contents/searchList`
    - **Category**: `c5` (축제/행사). 기존 `c4`(음식점)에서 수정됨.
    - **필터링**: 제주 공식 관광 포털의 정교한 카테고리(`c5`)와 태그 정보를 활용하여 실시간 축제 데이터를 수집합니다.
    - **상태 노출**: 축제 시작일과 종료일을 현재 날짜와 비교하여 '진행중' 또는 'D-Day' 라벨을 동적으로 계산하여 표시합니다.
    - **링크 연동**: 각 카드는 비짓제주의 상세 콘텐츠 페이지(`contentsid`)로 직접 연결됩니다.

## 🔍 습득물 정보 (Lost & Found)
- **제공처**: 경찰청 (LOST112)
- **방식**: 공공데이터포털 OpenAPI (`LosfundInfoInqireService`, `LosPtfundInfoInqireService`)
- **수집 파라미터**:
    - `N_FD_LCT_CD=LCP000`: 제주특별자치도 통합 검색 코드
    - `START_YMD` / `END_YMD`: 사용자 선택 기간 (기본 어제 ~ 오늘)
    - `PRDT_CL_CD_01`: 물품 대분류 필터링 (기존 `PRDCT_CL_CD`에서 수정한 정확한 파라미터)
- **데이터 병합 및 제한**: 
    - 경찰관서 습득물(`numOfRows=200`)과 포털기관 습득물(`numOfRows=200`)을 각각 호출하여 병합합니다.
    - **최대 조회 건수**: 한 번의 조회 시 각 API별로 200건씩, **총 400건**으로 제한되어 있습니다. 이는 과도한 API 호출 및 로딩 지연을 방지하기 위함입니다.
  - `LosPtfundInfoInqireService`: 포털 유관 기관(공항, 택시, 지하철 등)의 습득물을 조회합니다. (최신 적용)
  - **제주 전용**: 두 API 모두 지역 코드(`N_FD_LCT_CD=LCP000`)를 사용하여 제주도 전체 정보를 통합 검색합니다.


---
**주요 설정 파일**: `script.js` 내의 `CONFIG` 객체에서 API 키 및 엔드포인트 관리를 수행합니다.
